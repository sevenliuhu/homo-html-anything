【模板: 抖音短视频脚本 Deck】
【意图】生成抖音/快手风格的短视频分镜脚本 Deck。每页一个分镜, 包含: 画面描述 + 口播文字 + 时长标记 + 音效/BGM提示。适合直播预热、商品带货、剧情号、知识分享。

【布局 / 每页分镜】
- **画布**: 1080×1920 竖版, 安全区 (上下各留80px)
- **页眉**: 镜头编号 (01/08) + 时长 (0:03-0:08)
- **画面区**: 上半部分, 描述镜头画面 (简化线框图/emoji示意/CSS渐变) + 拍摄提示 (特写/全景/推拉)
- **口播区**: 下半部分, 口播文字 + 语气提示 (🎤正常 / 😂调侃 / 😱震惊)
- **音效标记**: 右上角小标签 (BGM: 轻快 / SFX: 叮咚 / VFX: 缩放)
- **商品关联**: 如带货, 标注商品名称 + 展示方式 (手持/桌面展示)

【设计细节】
- **配色**: 抖音风格 — 暗色底 `#161823` + 亮青 `#00f2ea` + 亮粉 `#fe2c55`
- **字体**: 中文 `Noto Sans SC`, 英文 `Inter`, mono数字 `JetBrains Mono`
- **排版**: 大字标题48px, 画面描述20px, 口播30px, 时间+编号mono 16px
- **动效**: 页面过渡 slide-up, 关键帧闪烁提示
- **底部**: 合集名称 + 页码 (如 "美妆带货 · 01/08")
- **限制**: 单文件HTML; 所有图片CSS占位; 单页deck可导出为脚本list
